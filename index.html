Rick and Morty Quotes Telegram Bot

What does it do?

Posts random provided quotes / images / gifs to Telegram Chats

You can easily add the Bot to your chats. Just add RickAndMortyQuoterBot.

Provides the following commands

  • /start to start the conversation, check if it is running.
  • /quote to get a random quote and /quote id to get a specific quote
  • /image to get a random image
  • /gif to get a random gif
  • /help to display the help dialogue
  • /info to get bot info

In group chats with multiple Bots it might be necessary to type /quote@RickAndMortyQuoterBot to send commands to the bot.

You can easily add the Bot to your chats. Just add D3LSBOT.

Feel free to fokk off / images / gifs :)

How do I use it?

Deploy the Code, provide your Telegram API Token and simply run with ruby main.rb

Can I adapt it?

Sure, change the quotes.json and images and gifs folders with your own and fire it up!
